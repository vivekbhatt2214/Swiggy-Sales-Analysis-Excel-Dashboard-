# Swiggy Sales Analysis Dashboard (Excel)

## Project Overview

This project analyzes Swiggy 2022 sales data using Microsoft Excel.

### Dashboard Features

- Total Sales Analysis
- Average Rating
- Average Order Value
- State-wise Sales Analysis
- Monthly Sales Trend
- Daily Sales Trend
- Weekly Sales Trend
- Food Category Analysis
- Top Cities Analysis

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Map Charts
- Conditional Formatting

## Key Metrics

- Total Sales: ₹53.01M
- Average Rating: 4.34
- Total Orders: 197.43K
